设置滤镜浓度

在美女秀场等应用场景里，滤镜浓度的要求会比较高，以便更加突显主播的差异。 我们默认的滤镜浓度是0.5，如果您觉得滤镜效果不明显，可以使用下面的接口进行调节。

参数：

strength 从0到1，越大滤镜效果越明显，默认值为0.5。
