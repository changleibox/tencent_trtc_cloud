enableAdjustRes property Null safety

bool enableAdjustRes
read / write

【字段含义】是否允许调整分辨率

【推荐取值】 - 手机直播建议选择 false。 - 视频通话模式，若更关注流畅性，建议选择 true，此时若遇到带宽有限的弱网，SDK 会自动降低分辨率以保障更好的流畅度（仅针对 TRTCVideoStreamTypeBig 生效）。 - 默认值：false。

【特别说明】若有录制需求，选择 true 时，请确保通话过程中，调整分辨率不会影响您的录制效果。
